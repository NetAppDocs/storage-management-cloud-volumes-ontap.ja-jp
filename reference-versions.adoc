= Cloud Volumes ONTAPデプロイメントでサポートされているONTAPバージョン
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NetApp Consoleを使用すると、Cloud Volumes ONTAPシステムを追加するときに、複数の異なるONTAPバージョンから選択できます。

ここに記載されている以外の Cloud Volumes ONTAP バージョンは、新規導入には使用できません。ここでのリリースのパッチまたは汎用（一般提供）バージョンは、導入に使用可能な基本バージョンを表します。利用可能なパッチの詳細については、各リリースの https://docs.netapp.com/us-en/cloud-volumes-ontap-relnotes/reference-new.html["バージョン付きリリースノート"^]を参照してください。

アップグレードの詳細については、link:task-updating-ontap-cloud.html#supported-upgrade-paths["サポートされているアップグレード パス"]を参照してください。
