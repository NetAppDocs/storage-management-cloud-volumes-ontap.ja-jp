=== Cloud Volumes ONTAP 9.16.1 GA

BlueXPを使用して、Azure および Google Cloud でCloud Volumes ONTAP 9.16.1 一般提供リリースを展開および管理できるようになりました。ただし、このバージョンは AWS でのデプロイおよびアップグレードには使用できません。
