=== 実行できないアクティビティ（タブ別）

次の表は、タブごとに制限されているアクティビティをまとめたものです。

[cols="1,4"]
|===
| タブ | 実行できないアクティビティ 


| 概要  a| 
* ボリュームの追加
* アグリゲートを追加
* System Manager を開きます
* オン / オフボタンを使用する
* パスワードを設定
* ONTAPバージョンを更新する
* ワークスペースからシステムを削除します
* システムを削除する
* * 今すぐアップグレード * を使用します
* すべての機能の鉛筆メニューから *System tags* を編集します。
* 全機能ペンシルメニューから *Scheduled downtime* を編集します
* 全機能ペンシルメニューから *Storage classes* を編集します
* 全機能の鉛筆メニューから*Instance type*を編集します。
* すべての機能のペンシルメニューから*充電方法*を編集します
* 全機能ペンシルメニューから *License type* を編集します
* すべての機能の鉛筆メニューから*書き込み速度*を編集します
* 全機能メニューの鉛筆アイコンから *Ransomware protection* を編集します。
* 全機能ペンシルメニューから *Support registration* を編集する
* 全機能ペンシルメニューから *WORM* を編集します
* 全機能ペンシルメニューから * Data tiering * を編集します
* 全機能メニューの鉛筆アイコンから *CIFS setup* を編集します。
* 全機能ペンシルメニューから*ルートテーブル*を編集します




| ボリューム  a| 
このタブではボリュームの更新は許可されておらず、ボリュームを追加することもできません。

* ボリュームをクローニングする
* ボリューム設定を編集
* ボリュームを削除する
* Snapshot コピーを作成する
* Snapshot コピーからリストアする
* ディスクタイプを変更する（詳細オプション）
* 階層化ポリシーを変更する（詳細オプション）




| アグリゲート  a| 
* アグリゲートを追加
* ボリュームの追加
* 容量の追加
* アグリゲートを削除する




| レプリケーション  a| 
* レプリケーションを解除
* レプリケーションの再同期
* 逆方向の再同期レプリケーション
* スケジュールを編集する
* 最大転送速度を編集します
* レプリケーションを更新
* レプリケーションを削除




| メインキャンバス  a| 
* システムを追加する
* システムを検出する（すべてのオペレーター向け）
* *コピーと同期*にドラッグ アンド ドロップを使用
* *ボリュームキャッシュ*にドラッグ アンド ドロップを使用
* *レプリケーション*にドラッグ アンド ドロップを使用
* ボリュームの追加
* *コストとパフォーマンスの最適化* にアクセスします
* System Manager を開きます
* 必要な CVO 権限が利用可能な場合にのみ Cloud Volumes ONTAP に *レプリケーション* を使用します（FSx for ONTAP またはオンプレミスシステムには適用されません）
* Cloud Volumes ONTAP に*ボリュームキャッシュ*を使用するのは、必要な CVO 権限が利用可能な場合のみです（FSx for ONTAP またはオンプレミスシステムには適用されません）
* システムを入力します
* AVS 管理ボタンを使用する




| ロールなし  a| 
ロールが割り当てられていない場合、すべての操作は無効になります。

* 作業環境にアクセスする
* レプリケーションへのアクセス
* Cloud Volumes ONTAP作成ウィザードを使用する
* レプリケーション作成ウィザードを使用する
* Cloud Volumes ONTAP を見つける


|===
